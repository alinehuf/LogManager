ESetting previousSetting,
int64 & nextId,
ESetting & nextSetting )
static

Creates a new config.json file with the next participant ID and the next experiment setting.

The participant ID is generated by incrementing the previous one via GetNextID(). The setting cycles through all ESetting values in order (wraps around using ESetting::Count).

This function should be called between each XP session to prepare for the next participant.

Parameters
previousUserIdID of the participant who just completed the XP.
previousSettingSetting used during the just-completed XP.
nextId[out] Generated ID for the next participant.
nextSetting[out] Setting to be used for the next participant.

Records the final survey answers in the currently open log file.

Writes a FINAL_SURVEY event containing all question-answer pairs as a nested JSON object. The log file must already be open (e.g., from a prior WriteLog_XPStart() call).

Parameters
gameTimeIn-game time (seconds) at the moment the survey was submitted.
questionsAndAnswersMap of question strings to their corresponding answer strings.

Records the start of an XP (experiment) run.

Creates and opens a new timestamped JSON log file, then writes an XP_START event containing the participant ID, current experiment setting, and initial head/player location.

File naming convention: YYYY-MM-DD_HHhMMmSSs_id<userId>_<setting>.json

Parameters
gameTimeIn-game time (seconds) at the moment of the XP start.
userIdUnique identifier of the current participant.
currentSettingCurrent experiment condition (table position).
startLocationWorld-space position of the player at XP start.

void ULogManagerBPFunctionLibrary::WriteLog_XPStop ( float gameTime)
static

Records the end of an XP (experiment) run.

Writes an XP_STOP event to the currently open log file, then closes it. The presence of this event in the file confirms that the XP session completed normally.

Parameters
gameTimeIn-game time (seconds) at the moment of the XP stop.
