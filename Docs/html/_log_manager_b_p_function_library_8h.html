Detailed Description

Blueprint Function Library exposing all LogManager logging operations to Blueprints.

ULogManagerBPFunctionLibrary is the Blueprint-facing API of the LogManager plugin. It provides a set of static functions callable from both C++ and Unreal Engine Blueprints, grouped under the "LOGLibrary" category in the Blueprint editor.

Each function delegates its work to FLogManagerModule, which manages the underlying third-party shared library responsible for writing JSON log files.

Typical usage flow in a Blueprint session:

  1. GetConfigFileInfos() — read the current participant ID and XP setting
  2. WriteLog_StartSurvey() — record the pre-XP survey in a separate file (only once per participant)
  3. WriteLog_XPStart() — open a new log file and record the XP start event
  4. WriteLog_CubeInteraction() / WriteLog_BoxEvent() — record in-session events
  5. WriteLog_XPStop() — record the XP end event and close the log file
  6. WriteLog_FinalSurvey() — append the final survey answers to the current file
  7. Loop step 3 to 6 with the same participant or CreateNewConfigFile() — increment the participant ID and cycle the setting
